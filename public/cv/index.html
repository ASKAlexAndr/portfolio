Возможности мультиязычного CV

  • Единый источник: Весь контент в одном HTML файле
  • Интерактивное переключение: Кнопки для смены языка
  • URL параметры: Поддержка прямых ссылок на язык (?lang=ru или ?lang=en)
  • Автоматическая генерация PDF: Два отдельных PDF файла из одного источника
  • Консистентное форматирование: Одинаковый стиль для обоих языков
  • Печать-оптимизировано: Скрытие элементов управления при печати

Как использовать

  • Просмотр: Откройте cv.html в браузере
  • Переключение языка: Используйте кнопки RU/EN в правом верхнем углу
  • Прямые ссылки: cv.html?lang=ru или cv.html?lang=en
  • Генерация PDF: Запустите npm run generate-pdf в терминале
